# Taj Electronics HTML Page

Files included:
- `index.html` — Full standalone HTML page using Tailwind CDN
- `cursor-prompt.txt` — Prompt you can paste into Cursor
- `/images` — Generated website images for Taj Electronics

## How to Use

Open `index.html` directly in your browser.

This version uses:
- Tailwind CSS CDN
- Google Fonts: Inter and Orbitron
- Local generated images
- Lightweight scroll reveal animation using JavaScript

## For React/Cursor

Use `cursor-prompt.txt` to generate a React version if needed.
